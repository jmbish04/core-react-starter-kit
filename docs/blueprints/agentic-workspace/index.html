LangGraph Orchestrator Pattern

Adapted from MAQuesada/Google-Workspace-Agent to run on Cloudflare Workers.

  • 1. The Planning Phase: Before any API call, the Agent creates a JSON Blueprint. This allows for validation (e.g., "7x7 Rule" in Slides) and structural integrity checks.
  • 2. Review Agents: A secondary agent loop that:
    • Fetches the raw HTML/content after generation.
    • Checks against best practices (accessibility, formatting).
    • Applies "Self-Healing" fixes before notifying the user.

PATTERN: PLAN -> EXECUTE -> REVIEW

PHASE 1: PLAN (JSON Blueprint)
PHASE 2: EXECUTE (Batch API)
PHASE 3: REVIEW (HTML Fetch & Fix)

Active Collaboration & Creation

  • 1. Resume Builder: Agent creates an outline, fills sections, and then reads back the HTML to ensure consistent header spacing.
  • 2. Presentation Architect: Agent generates a JSON plan for slides. It validates content density (7x7 rule) and ensures speaker notes are present.
  • 3. Sheet Architect: Creates multi-tab spreadsheets by defining schema first. Handles formula propagation via batchUpdate.

Context-Aware Drafting

Includes Domain-Graph extraction and Hierarchical Voice Tuning to match specific client relationships. Uses Vectorize to find previous threads with similar participants.
